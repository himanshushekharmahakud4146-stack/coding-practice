# Array Problem Patterns

Arrays are one of the most common data structures used in coding interviews.  
Many problems follow similar patterns. Understanding these patterns helps solve problems faster.

---

## 1. Array Traversal

The most basic pattern is iterating through an array.

This is used when we need to check or process every element.

Example:

```python
nums = [1, 2, 3, 4]

for num in nums:
    print(num)
```

Common techniques used in array problems:

1. Traversal
Loop through array elements.

2. Hash Set
Used to check duplicates quickly.

3. Two Pointer Technique
Useful for sorted arrays.

4. Tracking Min/Max
Used in stock profit problems.
