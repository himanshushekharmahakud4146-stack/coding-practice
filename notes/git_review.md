# Git Command Review

## Commands Reviewed

git status  
Checks the current state of the working directory and staging area.

git log --oneline  
Shows the commit history in a short format.

git add  
Stages changes to be committed.

git commit  
Records changes to the repository with a message.

git push  
Uploads commits from the local repository to GitHub.

git pull  
Fetches and merges changes from the remote repository.

git branch  
Lists or creates branches.

git checkout  
Switches between branches.

## Key Understanding

Git helps track code changes and maintain version history.  
Using Git daily improves project organization and collaboration skills.
