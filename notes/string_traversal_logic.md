# String Traversal Logic

## Key Patterns Learned

### 1. Basic Traversal
- Loop through characters
- Used for simple checks

---

### 2. Two-Pass Traversal
- First pass: store data (frequency)
- Second pass: apply condition

Used in:
- First Unique Character

---

### 3. Word Traversal
Steps:
1. Remove extra spaces
2. Split string into words
3. Reverse words
4. Join them back

Used in:
- Reverse Words in a String

---

## Important Learnings

- Strings are immutable
- Use split() and join() for efficiency
- Hashmaps are essential for frequency problems

---

## Mistakes to Avoid

- Ignoring edge cases (spaces, empty string)
- Overcomplicating logic
- Not practicing patterns

---

## Conclusion

String problems are mostly about:
- Traversal
- Pattern recognition
- Clean implementation
